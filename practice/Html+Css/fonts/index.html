Fonts and its Properties:

What is font?

Font is a property these are used to manipulate the text content on the element. These Properties controls the appearance of the text ,includeing its size,weight,style,family and more.

Font-family

its specifies the typeface for the text.

font-family:"Ariel","Helvetica","sans-serif";

Font-side:

Sets the size of the font or a text.

font-size:32px;

Font-weight:

It is defining the thickness of the font charactors

font-weight:normal/bold/bolder/lighter/100-900;

font-weight

Font-style:

Specifies the style of the font normal,italic,oblique

Font-variant:

Contols the use of the small-caps.

line-height:

Sets the amount of the space b/w the lines of the text.

font:[font-style][font-variant][font-weight]
