Topic about the attributr selector with Contains Specified value

Attribute Value contains selectors([attribute*="value"]):
Selects element whose attribute value contains the specified value.
Example: 
		       <a href="">Click me </a>
Syntax: a[href*="google"]{color:red;}
see me

Note:*:-Contains

Attribute Value contains selectors([attribute^="protocol"]):
Selects element whose attribute value contains the specified protocols like http,https.
Example: 
		       <a href="">Click me </a>
Syntax: a[href^="https"]{color:red;}
see me

Note:^:-Starts with

Attribute Value contains selectors([attribute$="Domain Name"]):
Selects element whose attribute value contains the specified Domain Name like .com,.org,.net.
Example: 
		       <a href="">Click me </a>
Syntax: a[href$=".com"]{color:red;}
see me

Note:$:-Starts with
