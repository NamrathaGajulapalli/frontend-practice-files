We are introducing our css grid here

Grid layout:

A grid is a 2-d layout system for web development.by this system content in rows and columns are simultaneously controled with the help of grid properties

Grid terminology:

Term Meaning
Grid Container That acts as a parent that established the grid context
Grid lines Grid lines are the dividing lines that form the structure of the grid,number form1 upward(both rows and columns)
Grid Track The space between two adjacent grid lines
Grid cells the smallest unit in grid that are intersection of one row track and one column track.
Grid Area Any rectagular space bounded by four grid lines or span of multiple
fraction(fr)unit A flexible unit meaning 'fraction available space eg:2fr,1fr

grid conainer poperties

Container properties are applied to the parent element (display:grid;) they define the structure ,sizing,alignmnet and spacing if the entire grid.

List of grid container properties

Display:

value description
grid Block-level grid container(takes fukk availablewidth)
inline-grid inline-level grid container(sits inline like a word)

grid-templet-columns

defines the number of columns in the grid and their widths. each value sepated by the space creates one column.

                        
	                       grid -template-column:200px 200px 200px;/1fr 1fr 1fr;

Grid-templete-rows

defines the number of columns in the grid and their widths. each value sepated by the space creates one column.

                        
	                       grid -template-row:200px 200px 200px;/1fr 1fr 1fr;

grid-templet-area

grid area is using a ASCII art-styles.each string representages a row each word in the string name the cell for the positions.

this is also called as "named layout"

                        
	                       grid -template-area:"header header header";
						                       "aside main main"
											   "footer footer footer"
