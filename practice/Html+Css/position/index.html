Demo on the position properties in the css.

What is the positions?

Position define how an element is placed in the website and it behaves in the relation to the other elemet and the browser window.The Positons property works together with the help of Top,left,Right,Bottomand Z-index

Types of positions:

  1. Static(default)
  2. Relative
  3. Absolute
  4. Fixed
  5. sticky

1.Static Position:

It is a Defalut Property for all the elements and follow the normal document flow and no change in the helping properties like Left,Right,Top,Bottom and Z-index

Position:static;

2.Relative Position:

Element stay in the normal flow its original position.Space to the element is still Preserved. It can accepts helping properties Top,Left,Right,Bottom and Z-index

Position:Relative;

3.Absolute Position:

Element is removed from the normal flow and the position relative to the nearset positioned ancestor.

Position:absolute;

4.Fixed Position:

The element is position relative is the view port and stay in the same place even when the scrolling. Removed from the document flow or normal flow and should have helping properties like Top,Left,Right,Bottom and Z-index

Position:fixed;

5.sticky Position:

it is the combination of Relative and Fixed and behaves like Relative untill a scroll point is reached and it fixed to the view port

Position:sticky;
