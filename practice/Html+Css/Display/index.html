flex container:

A flex container is the parent element in a flexbox layout, when we apply disply:flex; to an elment ,it becomes a flex container, and all its direction child or item becomes flex elements.

Flex container controles:
  • Direction of items
  • Arrangement
  • Spacing
  • Wrapping behaviours

List of flex container properties:

  1. flex-direction:values;(ex:row,column,row-reverse,column-reverse)
  2. justify-content:values;(ex:end,center,start,space-between,space-evenly,left,right,space-around)
    • justify-content: space-between → first item at start, last item at end, equal space only between items.
    • justify-content: space-around → equal space around items, but edges get half the space of the gaps.
    • justify-content: space-evenly → equal space between items and at both edges.
  3. align-item
    • align-items: flex-start → items align to the top/start of the cross-axis.
    • align-items: flex-end → items align to the bottom/end of the cross-axis.
    • align-items: center → items align at the center of the cross-axis.
    • align-items: stretch → items stretch to fill the cross-axis (default).
    • align-items: baseline → items align based on their text baseline.
  4. flex-wrap
    • flex-wrap: nowrap → all items stay in one line (may overflow).
    • flex-wrap: wrap → items move to next line when space is not enough.
    • flex-wrap: wrap-reverse → items wrap to next line in the reverse cross-axis direction.
  5. gap
    • gap: 20px → sets equal space between rows and columns.
    • gap: 10px 30px → 10px row gap, 30px column gap
    • row-gap: 15px → space only between rows.
    • column-gap: 25px → space only between columns.

List of flex item Properties:

  1. Order
  2. Flex-grow
  3. flex-shrink

  5. align-self

1.Order:

Change the visual order of a item without changeing the html structure.

				 Syntax:
				 
				    .item1{
					 order:2;
					 }
					.item2{
				     order:1
				     }
  • Default value of order is "0"

2.Flex-grow:

controles how much a flex item grows relative to other when extra space is available.
				 Syntax:
				 
				    .item1{
					 flex-grow:2;
					 }
					.item2{
				     flex-grow:1
				     }
  • Default value of flex-grow is "0"(does not grow)

3.Flex-shrink:

controles how much a flex item shrinks when the space is limited. relative to other when extra space is available.
				 Syntax:
				 
				    .item1{
					 flex-shrink:2;
					 }
					.item2{
				     flex-shrink:10%;
				     }
  • Default value of flex-shrink is "0"(prevents shrink)

4.Flex-basis:

Specifies the initial desize of the flex item before space distribution.
				 Syntax:
				 
				    .item1{
					 flex-basis:200px;
					 }
					.item2{
				     flex-basis:100px;
				     }
  • canbe in px,%,em or rem.
  • the default value is is auto.

5.align-self:

Overrides the container's align-items property for individual items along the cross axis.
				 Syntax:
				 
				    .item1{
					 aline-self:flex-end/flex-start/center/auto/stretch/baseline and so on;
					 }
					.item2{
				     flex-shrink:10%;
				     }
values Description
auto Default(follows container)
flex-start start of the cross axis
flex-end end of the cross axis
center center of the cross axis
stretch stretch to full
baseline align based on the text bese line
  • Default value of flex-shrink is "0"(prevents shrink)
